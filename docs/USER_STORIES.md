## User Stories

### US-1: Add Book
As a librarian, I want to add a book so that it can be stored and managed in the system.

### US-2: Borrow Book
As a librarian, I want to borrow a book so that its availability status is updated.

### US-3: Return Book
As a librarian, I want to return a borrowed book so that it becomes available again.

### US-4: Generate Report
As a librarian, I want to generate a report so that I can view the current status of all books.
